oulu-rails
==========

sass tools for machida.

Docs
----

-	[index](/docs/index.md)

Installation
------------

-	Add `source 'https://rails-assets.org'` at Gemfile.
-	`$ bundle`
-	Add `@import oulu` at app/assets/stylesheets/application.css.sass
